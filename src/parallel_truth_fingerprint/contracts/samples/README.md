# Payload Contract Samples

These files are copied from `docs/input/` and preserved here as implementation-local reference artifacts.

They are source-of-truth examples for later contract stories:

- `hart_payload_sample.txt`: raw edge acquisition HART-style payload
- `unified_hart_payload_sample.txt`: post-consensus unified payload example

Do not reinterpret or flatten these samples during bootstrap. Later stories should use them to define typed contract models and payload-driven observability.
